Proposito del Sitio


La idea principal de este sitio web es funcionar como una herramienta orientativa que permita a los usuarios aprender sobre el lenguaje de programación web conocido como HTML, para esto de abordaran los conceptos básicos desde un inicio para ir escalando en el estudio y uso de herramientas más complejas; el objetivo es facilitar la obtención de información para que los estudiantes puedan aplicar las temáticas expuestas en la construcción de sus sitios Web.


¿Que se encontrara en el sitio?


Dentro de las diferentes pestañas que ofrece el menú principal es posible encontrar espacios donde se exponen de forma detallada las etiquetas básicas usadas en HTML, temas como que son los atributos y los valores. También abran recursos digitales que apoyaran en la construcción de los sitios web en los que se este trabando, para esto se aprovechan las herramientas digitales que puedan llegar a cumplir la función de tutorial.


Mas antes de abordar las temáticas principales o explorar los recursos digitales es necesario preguntarse ¿Qué es HTML?, la respuesta no es tan compleja se trata de un lenguaje de programación enfocado completamente al desarrollo de páginas web.



¿Que es una etiqueta?


Según Ceyla Luna las etiquetas son HTML es un leguaje de marcas o etiquetas, lo que quiere decir que los elementos que conforman la pagina web so descritos a partir de palabras especiales que marcan el inicio o final, a esto se le denomina etiqueta.



¿Que es una Atributo?


Son valores adicionales que ayudan a especificar un elemento y definir su comportamiento dentro de la página, estos van dentro de las etiquetas de apertura esto con la idea de complementar la información del elemento que use quiere representar.



¿Que son los valores?


Los valores son las propiedades que se le pueden asignar a un elemento dentro de la página, mientras que el atributo define el comportamiento de la etiqueta el valor se encarga de especificar las características del elemento.



Etiquetas Basicas


Dentro de la sintaxis del HTML hay componentes básicos que conforman una pagina web, a estos los denominamos etiquetas, estas cumplen la función de indicarle al navegador donde inicia y donde termina un elemento. Dentro de la estructura básica encontramos tras áreas:

  • Etiquetas de apertura: Aquí se indica donde inicia un elemento y se representa con los símbolos <>.
  • Contenido: Es el producto final el cual puede ver el usuario.
  • Etiquetas de cierra: Funciona de la misma forma que las etiquetas de cierra mas estas cumplen la función de indicar el final de un elemento se simbolizan con los signos <> y un / en su interior.


Con la estructura anterior ahora es posible observar que etiquetas son las mas frecuentes dentro de un documento HTML.

  • body: Se usa para definir el contenido visible del documento HTML.
  • head: Se usa para definir la información, es decir elementos como el título de la página, estilos, entro otros atributos que no son notorios para el usuario promedio.
  • div: Permite agrupar elementos con el fin de poder estructurar de mejor manera las secciones.
  • a: Facilita el crear enlaces que redireccione a una página web.
  • strong: Ayuda a destacar una palabra o una frase de un texto largo.
  • br: Se le indica al navegador que existe un salto de línea.
  • h1, h2. . .h6: Define los diferentes niveles de títulos, ayudando así al navegador a marcar la importancia de cada contenido.
  • img: Ayuda a insertar una imagen en la página, apoyándose en el atributo SRC se pude definir en donde se debe mostrar la imagen
  • ul y li: Permite crear listas
  • p: Agrupa los textos en un párrafo
  • span: Permite personalizar una parte del texto expuesto en la página.
